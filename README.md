# 🎥 AI YouTube Video Analyzer

Analyze any YouTube video using AI in seconds.

🔗 **Live App:** https://ai-youtube-video-analyzer-xxxx.streamlit.app

---

## 🚀 Features
- 📌 Video summary
- 🔍 Key insights extraction
- ⏱️ Timestamp-based breakdown
- ⚡ Fast analysis using Groq LLMs
- 🤖 Agent-based architecture (Agno)

---

## 🧠 How it works
This project uses an **Agno AI agent** powered by **Groq LLMs** to:
- Fetch video data using YouTube tools
- Understand content semantically
- Generate structured insights
